Steps used for compile the app
==============================

* Open "Developer Command Prompt for VS2013" in directory
  C:\Program Files (x86)\Microsoft Visual Studio 12.0\Common7\Tools\Shortcuts

* Add CMake to the PATH:
    > PATH=%PATH%;"C:\Program Files (x86)\CMake 2.8\bin"

* Go to project directory:
    > cd \Users\myuser\testgles

* Create and go to the build directory
    > mkdir build
    > cd build

* Generate project files for build
    > cmake -G "NMake Makefiles" ..

* Build projects (this copy needed dll's to the build directory)
    > nmake

* Run
    > emptySDL
